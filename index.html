IT'S ALIVE!

Monday, October 22nd, 2018

Just in time for Halloween, I've created my own monster - my first website designed (almost) from scratch with HTML, CSS, and lots of Googling, as opposed to using a website building service like Squarespace or Freewebs.

This website was started from the HTML code found in the How TO - Make a Website tutorial on w3schools.com. The current design is heavily inspired by nin.com. I'm currently challenging myself to recreate the Nine Inch Nails main page to help me learn HTML and CSS, and after I get a little more comfortable, I'll design something original.

Though, I love this minimalistic style, so my future website will likely still have some similarities.
